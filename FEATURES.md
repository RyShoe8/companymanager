# Nucleas Feature List

Comprehensive feature list organized by site sections for marketing purposes.

## Projects

- **Custom Project Colors** - Color-code projects for easy visual identification
- **Project Tasks** - Break down projects into manageable tasks with clear milestones and deliverables
- **Time Estimation** - Estimate hours for each project to better plan resources and timelines
- **Team Assignments** - Assign team members to projects and track their workload across all assignments
- **Status Tracking** - Monitor project status: Planning, In Development, In Review, or Completed
- **Timeline Management** - Set start and end dates for better planning
- **Plan → Build → Run Phases** - Projects move through phases; the interface adapts to each phase
- **Project Marketing Stack** - Configure hosting, analytics, domains, design, docs, billing, and marketing tools
- **Project Tech Stack** - Document and display technology stack used for each project
- **Project Social Links** - Add and manage social media links for projects
- **Project Logos** - Upload and display custom project logos
- **Resource Planning** - See how projects impact team member capacity
- **Flexible Tasks** - Define as many tasks as needed for each project

## Tasks

- **Task Breakdown** - Define detailed tasks within projects with clear deliverables
- **Task Status Tracking** - Monitor individual task progress and completion
- **Task Recurrence** - Set up recurring tasks for repeated work
- **Task Linked Assets** - Link relevant assets and resources to specific tasks
- **Task Assignments** - Assign tasks to specific team members
- **Task Time Estimation** - Estimate hours for individual tasks
- **Task Milestones** - Set milestones within tasks to track progress
- **Inline Task Management** - Manage tasks directly within project views

## Content

- **Content Items** - Create and manage content items with detailed information
- **Content Targeting** - Define target audience, platform, and distribution channels
- **Content Channels** - Organize content by channel or platform
- **Content Linked Assets** - Link assets, documents, and resources to content items
- **Content Scheduling** - Schedule content items for specific dates and times
- **Content Creation Modal** - Quick interface for creating new content items
- **Content Detail Modal** - Detailed view and editing of content items
- **Content Item Assets Section** - Dedicated section for managing content-related assets
- **Content Channel Filter** - Filter content by channel for focused viewing

## Meetings

- **Meeting Creation** - Create and schedule meetings with team members
- **Meeting Agenda** - Define and manage meeting agendas
- **Calendar View** - Visual calendar view for scheduling and viewing meetings
- **Meeting Availability** - Set and manage availability for meeting scheduling
- **Meeting Project Insights** - See project context and relevant information in meetings
- **Meeting Join Call** - One-click join button for video calls
- **Meeting Detail View** - Comprehensive view of meeting details, agenda, and participants
- **Scheduling Panel** - Dedicated panel for managing meeting schedules
- **Scheduling Calendar Bar** - Quick access calendar bar for scheduling
- **Meeting Agenda Rows** - Detailed agenda item management within meetings

## Tools (Nucleas OS)

- **OS Modules** - Modular interface with Agenda, Projects, Schedule, and Voice modules
- **Floating Windows** - Draggable and resizable floating windows for multitasking
- **Command Palette** - Quick access to commands and actions via keyboard
- **Voice Commands** - Control the interface using voice commands
- **Voice Intent Parsing** - AI-powered understanding of voice commands (with OpenAI)
- **Voice Overlay** - Visual interface for voice interactions
- **Screenshot Capture** - Capture screenshots directly within the application
- **Screenshot Gallery** - View and manage captured screenshots
- **Recording Capture** - Record screen activity and audio
- **Recording Management** - Save, organize, and access recordings
- **Asset Linking** - Link assets to projects, tasks, and content items
- **Smart Buttons** - Launch hosting, analytics, docs, and more in one click from each project
- **Popout Windows** - Open modules in separate windows for focused work

## Employees & Utilization

- **Capacity Tracking** - Set and track weekly hours for each team member to understand availability
- **Workload Visibility** - See how projects and tasks impact each team member's workload
- **Role Management** - Assign roles (Administrator, Manager, User) with appropriate permissions
- **Assignment Tracking** - See all projects and tasks assigned to each team member
- **Weekly Hours** - Set and track weekly capacity for each employee
- **Employee Types** - Categorize employees as full-time, part-time, or contractor
- **Role-Based Access** - Assign roles with appropriate permissions and access levels
- **Workload Analysis** - See how assignments impact team member capacity
- **Team Overview** - Get a complete view of your team's capacity and assignments
- **Employee Sidebar** - Quick access to employee information and assignments

## Efficiency Features

- **Command Palette** - Quick access to all commands and actions via keyboard shortcut
- **Voice Commands** - Execute actions using natural voice commands
- **AI Intent Parsing** - Intelligent understanding of user intent for voice and command palette (with OpenAI)
- **Screenshot Tools** - Capture and annotate screenshots without leaving the app
- **Recording Tools** - Record screen activity and audio for documentation
- **Real-time Search** - Find projects, tasks, assets, and content instantly
- **Advanced Filtering** - Filter by type, status, assignee, phase, and more
- **Smart Buttons** - Launch external tools and services in one click
- **Quick Project Form** - Rapid project creation with minimal friction
- **Quick Content Creation** - Fast content item creation with pre-filled information
- **Drag-and-Drop Scheduling** - Easily move items between time horizons
- **Keyboard Shortcuts** - Efficient navigation and action execution
- **Phase Filtering** - Quickly switch between Plan, Build, and Run phases
- **Lens Views** - Different perspectives on your workspace (Schedule, Agenda, etc.)

## Organizational Features

- **Organization Setup** - Guided setup for new organizations
- **Organization Branding** - Custom branding for your organization
- **Organization Modal** - Manage organization settings and preferences
- **Multi-Project Management** - Manage multiple projects from a single dashboard
- **Centralized Asset Repository** - Single source of truth for all tools, documents, and resources
- **Asset Categorization** - Organize assets by type: Tools, Documents, Links, Screenshots, Files
- **Asset Search & Filter** - Quickly find assets with powerful search and filtering
- **Project & Task Links** - Link assets to specific projects and tasks for context
- **Team Collaboration** - Built-in collaboration features for teams
- **Comments & Discussions** - Comment on projects, tasks, and content items
- **Checklists** - Create and manage checklists for projects and tasks
- **Email Digests** - Configure email digests for team updates
- **Invitations** - Invite team members to join your organization
- **Profile Management** - Manage user profiles and preferences
- **Feedback System** - Collect and manage feedback from users
